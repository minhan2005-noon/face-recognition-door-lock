# Dashboard

Dashboard là giao diện web quản trị hệ thống khóa cửa.

## Cần làm

- Hiển thị trạng thái thiết bị.
- Quản lý người dùng được phép.
- Xem lịch sử ra vào.
- Gửi lệnh khóa/mở khóa nếu admin có quyền.

## Cài đặt

- Cài Node.js LTS.
- Chạy `npm install` sau khi có dependencies.
- Cấu hình endpoint backend bằng biến môi trường.

## Chạy dự kiến

- Development: `npm run dev`.
- Build: `npm run build`.
