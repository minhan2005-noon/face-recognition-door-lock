# Firmware Arduino

Thư mục này dành cho biến thể firmware dùng Arduino IDE hoặc Arduino CLI.

## Cần làm

- Thêm sketch trong `src/`.
- Đặt header dùng chung trong `include/`.
- Cấu hình chân relay/servo và giao tiếp serial/network.

## Cài đặt

- Cài Arduino IDE hoặc Arduino CLI.
- Cài board package phù hợp với thiết bị.
- Cài các library cần thiết qua Library Manager.

## Chạy

- Mở sketch trong Arduino IDE.
- Chọn board và port.
- Verify và Upload.
