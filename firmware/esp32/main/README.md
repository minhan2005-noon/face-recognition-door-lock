# ESP32 Main

Đặt file chương trình chính của ESP32 tại đây.

## Cần làm

- Khởi tạo camera hoặc module giao tiếp.
- Kết nối Wi-Fi.
- Lắng nghe lệnh mở khóa.
- Điều khiển relay/servo.
- Gửi trạng thái về backend.

## Gợi ý file

- `main.cpp`: điểm vào chương trình PlatformIO.
- `config.h`: cấu hình chân và endpoint.
