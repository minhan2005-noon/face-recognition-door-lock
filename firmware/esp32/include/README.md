# ESP32 Include

Đặt các file header dùng chung cho firmware ESP32.

## Cần làm

- Định nghĩa cấu hình chân.
- Định nghĩa kiểu dữ liệu trạng thái khóa.
- Khai báo hàm giao tiếp network, điều khiển khóa và camera.

## Lưu ý

- Không đặt secret thật vào đây nếu repository được chia sẻ.
- Dùng file mẫu riêng cho cấu hình nhạy cảm.
