# Dataset người dùng được phép

Đặt ảnh của người được phép mở khóa tại đây.

## Cấu trúc gợi ý

```text
authorized_users/
  user_001/
    image_001.jpg
    image_002.jpg
  user_002/
    image_001.jpg
```

## Cần làm

- Đặt tên thư mục ổn định cho từng người.
- Dùng ảnh có khuôn mặt thật rõ.
- Loại ảnh mờ, ngược sáng hoặc có nhiều khuôn mặt nếu pipeline chưa hỗ trợ.
