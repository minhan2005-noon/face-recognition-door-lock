# Models

Thư mục này chứa model đã huấn luyện và nhãn người dùng.

## File dự kiến

- `face_recognition_model.onnx`: model ONNX nếu dùng.
- `labels.json`: ánh xạ ID/label với người dùng.

## Lưu ý

- File model có thể lớn, cần cân nhắc dùng Git LFS.
- Không commit dữ liệu nhạy cảm nếu model có thể suy ngược danh tính.
