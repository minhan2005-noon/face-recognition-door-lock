# Routes

Routes định nghĩa các endpoint REST API.

## Cần làm

- `/api/users`: quản lý người dùng.
- `/api/accessLogs`: xem lịch sử ra vào.
- `/api/lock`: gửi lệnh khóa/mở khóa.
- `/api/devices`: quản lý thiết bị mở khoá
- `/api/health`: xem trạng thái thiết bị có đang on hay off
- `/api/recognitionEvents`: nhận kết quả nhận diện khuôn mặt từ AI/camera
- Gắn middleware xác thực cho endpoint nhạy cảm.
