# Tests

Thư mục này chứa kiểm thử cho từng phần của hệ thống.

## Cần làm

- `unit/`: test hàm/module nhỏ.
- `integration/`: test backend, AI model và firmware gateway.
- `hardware/`: test với thiết bị thật.

## Cài đặt

- Backend có thể dùng Jest/Vitest.
- Python AI có thể dùng pytest.
- Firmware có thể dùng PlatformIO test nếu phù hợp.
