# Scripts

Thư mục script hỗ trợ cài đặt, deploy và backup.

## Cần làm

- `setup.sh`: cài đặt dependency và tạo môi trường.
- `deploy.sh`: build/deploy các thành phần cần thiết.
- `backup.sh`: sao lưu database, model và cấu hình.

## Cài đặt

- Cấp quyền chạy khi script đã có nội dung: `chmod +x scripts/*.sh`.
- Kiểm tra trên môi trường development trước khi chạy production.
