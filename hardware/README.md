# Hardware

Thư mục này chứa tài liệu phần cứng của hệ thống khóa cửa.

## Cần làm

- Ghi danh sách linh kiện trong `components/`.
- Vẽ sơ đồ nối dây trong `wiring/`.
- Lưu schematic và PCB nếu có.
- Ghi rõ nguồn điện và cảnh báo an toàn.

## Cài đặt/lắp mạch

- Kiểm tra điện áp của relay/servo/khóa điện.
- Dùng nguồn riêng cho khóa nếu dòng lớn.
- Nối chung GND giữa board điều khiển và module liên quan.
